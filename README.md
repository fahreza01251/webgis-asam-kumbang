# WebGIS Pemetaan Jalan Rawan

## Kelurahan Asam Kumbang, Kec. Medan Selayang, Kota Medan

Sistem WebGIS berbasis website untuk memetakan lokasi
jalan gelap dan rusak di Kelurahan Asam Kumbang.

## Teknologi

- PostgreSQL + PostGIS (Supabase)
- Leaflet.js
- HTML + CSS + JavaScript
- QGIS
- Netlify

## Cara Menjalankan

1. Clone repository ini
2. Salin config.example.js menjadi config.js
3. Isi config.js dengan URL dan Key Supabase kamu
4. Buka index.html menggunakan Live Server

## Fitur

- Peta interaktif titik jalan rawan
- Input data lapangan dengan GPS
- Dashboard statistik Admin
- Kelola data (edit & hapus)
- Manajemen pengguna
- Analisis spasial PostGIS
